# djangonx
A sample django project configured with nginx in a Debian Stretch VM.


### Techs
1. Debian Stretch VM
2. Python/Django
3. nginx


### Installed
1. Django
2. SQLite
3. django-allauth


### Features
1. User can register
2. User can login
3. User can view profile
4. User can go to the django admin
